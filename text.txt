commond lits:

cd <directory> : change work directory
dir <directory>  : list files and directories
clear : clear the terminal
envrion : list all environment variable
echo <word> : print words into terminal
pause : pause untill press enter
help : dispaly the help
 exit : exit myshell

<directory of program>  <args list> : execute outernal program

<command> > <file> : redirect stdout to file
<command> < <file> : redirect stdin from file
